# BasitBorsa

**Borsayı anlamanın en basit yolu.**

BasitBorsa, yeni başlayanların borsayı sade ve anlaşılır şekilde öğrenmesi için geliştirilmiş yapay zekâ destekli bir demo eğitim platformudur.

Kullanıcılar seçili BIST hisselerini inceleyebilir, grafik hareketlerinin hikâyesini görebilir, temel finansal kavramları öğrenebilir ve sanal portföy ile risksiz pratik yapabilir.

> Bu proje yatırım tavsiyesi vermez. Gerçek alım-satım işlemi veya gerçek para kullanımı içermez.

## Özellikler

- Seçili BIST hisselerini listeleme
- Basit hisse detay sayfası
- Grafik hareketleri için açıklamalar
- Yapay zekâ destekli finansal öğrenme
- Sanal portföy ile demo pratik
- Başlangıç seviyesi finansal kavramlar
- Seed/fallback veri desteği

## Teknolojiler

- React
- Vite
- Tailwind CSS
- Java
- Spring Boot
- PostgreSQL
- Gemini API

## Kurulum

git clone https://github.com/kullanici-adi/basitborsa.git
cd basitborsa
Backend
cd backend
mvn spring-boot:run
Frontend
cd frontend
npm install
npm run dev

Uyarı
BasitBorsa bir yatırım platformu değildir.
Gösterilen bilgiler eğitim ve demo amaçlıdır. Veriler gecikmeli, gün sonu, cache’lenmiş veya demo veri olabilir.
